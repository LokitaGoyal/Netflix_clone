# 🎬 Netflix Clone

A responsive Netflix landing page clone built using **HTML, CSS, and JavaScript**. This project recreates the look and feel of Netflix's homepage, including responsive layouts, interactive FAQ sections, and a language dropdown.

## 🚀 Features

- 🎨 Modern Netflix-inspired UI
- 📱 Fully responsive design
- 🎥 Background videos and animations
- ❓ Interactive FAQ accordion
- 🌐 Language dropdown menu
- 📧 Email subscription section
- 💻 Clean and organized code

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/LokitaGoyal/Netflix-Clone.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or additional dependencies are required.
